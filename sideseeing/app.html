MultiSensor Data Collection App

MultiSensor Data Collection is a mobile app that seamlessly gathers data from various sensors, including accelerometer, gyroscope, magnetometer, GPS, video camera, and audio, allowing you to create a comprehensive dataset locally and effortlessly transmit it to the cloud for further analysis and storage. The application's source-code is publicly available here.

Installation

The first thing you need to do is provide us with the email associated with your Google account, which will be added to a list of internal users. As the app is currently in the testing stage, this email is the only means by which you will be able to install it. Check the complete instructions here.

Data Characteristics

The recorded data, located in the Download/MSDC folder on the device, consists of up to seven files, depending on the device's capabilities. The following table describes each of these files.

File Description
consumption.csv File containing data related to power consumption
gps.csv File containing GPS data
metadata.json File containing metadata.
sensors.one.csv File containing data from one-axis sensors
sensors.three.csv File containing data from three-axis sensors
sensors.three.uncalibrated.csv File containing data from three-axis sensors uncalibrated
video.mp4 MP4 video file
